## Instalações
### Requerimentos
  *  `linux OS`
  * `python3`
  * `pip3`


## How to run 

`python3 se7e-_-db.py` to open interface


## Arguments to use
### optional arguments
    Ctrl+h - clear typos
    Ctrl+c or exit - to close the script
    Use ↑ and ↓  - arrows to move the cursor
    Space - to select options
    Enter - to initiate action
